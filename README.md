## Descripción
Este proyecto consiste en el desarrollo de una calculadora como trabajo práctico de la materia Ingeniería de Software I.

## Alcance inicial
En una primera etapa, la calculadora incluirá:

- suma
- resta
- multiplicación
- división

## Objetivo
Desarrollar una solución funcional aplicando un proceso de trabajo organizado, contemplando relevamiento de requerimientos, diseño, implementación, pruebas y presentación final.
